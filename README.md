# 🛠️ Patrones de Diseño (POO II)

¡Hola! 👋 Este repositorio contiene los ejercicios resueltos de las semanas 4 a 7 del curso de **Programación Orientada a Objetos II**.
---

## 💻 Lenguajes Utilizados
* **Python 3.x**
* **C++**

---

## 📂 Contenido del Repositorio

### 1️⃣ Patrones Creacionales (Semana 4 y 5)
*Objetivo: Controlar cómo se crean los objetos.*
* **Singleton (C++)**: Como una "llave única" para la configuración de un sistema.
* **Factory (Python)**: Una "fábrica" de transportes que decide si necesitas un Camión, Barco o Avión.
* **Builder (Python)**: Como armar un menú de comida rápida paso a paso (hamburguesa, bebida, papas).

### 2️⃣ Patrones Estructurales (Semana 6)
*Objetivo: Organizar clases y objetos para que trabajen bien juntos.*
* **Adapter (Python)**: Un "conversor de enchufes" para que una pasarela de dólares acepte soles.
* **Decorator (Python)**: Añade "capas de seguridad" o validaciones a funciones matemáticas.
* **Composite (C++)**: Estructura de carpetas y archivos (jerarquías tipo árbol).

### 3️⃣ Patrones de Comportamiento (Semana 7)
*Objetivo: Mejorar la comunicación entre los objetos.*
* **Observer (Python)**: Un sistema de notificaciones de Chat en tiempo real.
* **Strategy (C++)**: Permite cambiar el método de ordenamiento (Burbuja o QuickSort) al instante.
* **Command (Python)**: El botón "Deshacer" (Undo) para un editor de texto.
